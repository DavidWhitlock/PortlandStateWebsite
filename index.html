In Contemporary Software Development with Java and Android we'll learn about and apply tools techniques used by Software Developers all over the world every day to quickly deliver high-quality software that is robust enough to meet today's needs and flexible enough to evolve to meet tomorrow's needs. You'll see how code is managed with version control, leverage unit tests to continuously demonstrate code quality, refactor code to improve it, build a small device application with Android, and experience developing software individually, in pairs, and with a small group.

The textbook for the course is The Art of Agile Development, Second Edition by James Shore. While the textbook covers many of the topics that are discussed in class, I do not "follow the book" per se.
