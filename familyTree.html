Family Tree Applet

Here is a Java Applet that uses components of the Family Tree GUI. Note that the XML files are loaded from URLs.

A couple of notes:

  • You need the Java Plug-In to view this applet that comes with the JDK and is probably already installed with your web browser
  • If suggest that you write an APPLET tag and then convert your HTML file with Sun's converter.
